Persetujuan Cookie EU

Untuk menggunakan situs web ini, kami menggunakan Cookies dan mengumpulkan beberapa Data. Untuk mematuhi GDPR EU, kami memberi Anda pilihan apakah Anda mengizinkan kami untuk menggunakan Cookies tertentu dan mengumpulkan beberapa Data.

Data Penting

Data Penting diperlukan untuk menjalankan Situs yang Anda kunjungi secara teknis. Anda tidak dapat menonaktifkannya.

- Session Cookie: PHP menggunakan Cookie untuk mengidentifikasi sesi pengguna. Tanpa Cookie ini, Situs Web tidak akan berfungsi.

- XSRF-Token Cookie: Laravel secara otomatis menghasilkan "token" CSRF untuk setiap sesi pengguna aktif yang dikelola oleh aplikasi. Token ini digunakan untuk memverifikasi bahwa pengguna yang diautentikasi adalah yang benar-benar membuat permintaan ke aplikasi.
