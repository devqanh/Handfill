Consentement Cookie de l'UE

Pour utiliser ce site web, nous utilisons des Cookies et collectons certaines Données. Pour être conforme au GDPR de l'UE, nous vous donnons le choix de nous autoriser à utiliser certains Cookies et à collecter certaines Données.

Données Essentielles

Les Données Essentielles sont nécessaires pour faire fonctionner techniquement le Site que vous visitez. Vous ne pouvez pas les désactiver.

- Session Cookie: PHP utilise un Cookie pour identifier les sessions utilisateur. Sans ce Cookie, le Site Web ne fonctionne pas.

- XSRF-Token Cookie: Laravel génère automatiquement un "token" CSRF pour chaque session utilisateur active gérée par l'application. Ce token est utilisé pour vérifier que l'utilisateur authentifié est bien celui qui effectue réellement les requêtes à l'application.
