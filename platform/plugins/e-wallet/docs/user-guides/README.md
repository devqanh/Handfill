## How to Use These Guides

### First Time Users

1. Start with the appropriate guide (Customer or Admin)
2. Read the "Getting Started" section
3. Follow step-by-step instructions
4. Refer to FAQ for quick questions

### Experienced Users

1. Use the Quick Reference for fast lookup
2. Check FAQ for specific questions
3. Refer to full guides for detailed procedures

### Training New Staff

1. Have them read the Admin Guide
2. Walk through common tasks together
3. Keep Quick Reference handy
4. Review FAQ for common questions
